# Text to QR Code Generator

A simple web application that converts any text or URL into a QR code instantly.  
Built using JavaScript as a beginner-friendly project.

## Features
- Convert text or links into a QR code
- Instant QR code generation
- Clean and simple user interface
- Works directly in the browser (no installation required)

## Built With
- HTML
- CSS
- JavaScript
- QR Code API / Library

## How to Use
1. Enter text or a URL in the input box
2. Click the "Generate QR Code" button
3. The QR code will be displayed instantly

## How to Run the Project
1. Clone the repository
2. Open `index.html` in your browser

- Handling user input and button events
- Working with external APIs / libraries
- Structuring a simple frontend project
